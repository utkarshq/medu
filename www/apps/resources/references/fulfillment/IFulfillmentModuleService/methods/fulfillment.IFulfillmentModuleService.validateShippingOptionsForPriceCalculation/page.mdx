# validateShippingOptionsForPriceCalculation - Fulfillment Module Reference

This documentation provides a reference to the `validateShippingOptionsForPriceCalculation` method. This belongs to the Fulfillment Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

This method checks whether a shipping option can have calculated price.

## Example

```ts
const isValid =
  await fulfillmentModuleService.validateShippingOptionsForPriceCalculation(
    [
      {
        provider_id: "webshipper",
        price_type: "calculated",
      },
    ]
  )
```
