# updateTaxRegions - Tax Module Reference

This documentation provides a reference to the `updateTaxRegions` method. This belongs to the Tax Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

## updateTaxRegions(data, sharedContext?): Promise&#60;[TaxRegionDTO](../../../interfaces/tax.TaxRegionDTO/page.mdx)&#62;

This method updates a tax region.

### Example

```ts
const taxRegion = await taxModule.updateTaxRegions({
  province_code: "be",
})
```
