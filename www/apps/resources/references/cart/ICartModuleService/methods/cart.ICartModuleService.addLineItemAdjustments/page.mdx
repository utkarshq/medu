# addLineItemAdjustments - Cart Module Reference

This documentation provides a reference to the `addLineItemAdjustments` method. This belongs to the Cart Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

## addLineItemAdjustments(data): Promise&#60;[LineItemAdjustmentDTO](../../../interfaces/cart.LineItemAdjustmentDTO/page.mdx)[]&#62;

This method adds line item adjustments to line items.

### Example

```ts
const lineItemAdjustments =
  await cartModuleService.addLineItemAdjustments([
    {
      item_id: "caliadj_123",
      code: "50%OFF",
      amount: 3000,
    },
    {
      item_id: "caliadj_321",
      code: "10%OFF",
      amount: 3000,
    },
  ])
```
