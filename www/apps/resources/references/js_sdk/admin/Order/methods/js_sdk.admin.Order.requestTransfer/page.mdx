# requestTransfer

This method requests an order transfer. It sends a request to the
[Request Order Transfer](https://docs.medusajs.com/api/admin#orders\_postordersidrequesttransfer)
API route.

## Example

```ts
sdk.admin.order.requestTransfer("order_123", {
  customer_id: "cus_123",
  internal_note: "Internal note",
})
.then(({ order }) => {
  console.log(order)
})
```
