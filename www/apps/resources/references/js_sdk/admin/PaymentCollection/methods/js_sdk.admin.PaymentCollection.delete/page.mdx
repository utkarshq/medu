# delete

This method deletes a payment collection. It sends a request to the
[Delete Payment Collection](https://docs.medusajs.com/api/admin#payment-collections\_deletepaymentcollectionsid)
API route.

## Example

```ts
sdk.admin.paymentCollection.delete("paycol_123")
.then(({ deleted }) => {
  console.log(deleted)
})
```
