# updateProducts

This method manages the products of a collection to add or remove them. It sends a request
to the [Manage Products](https://docs.medusajs.com/api/admin#collections\_postcollectionsidproducts)
API route.

## Example

```ts
sdk.admin.productCollection.updateProducts("pcol_123", {
  add: ["prod_123"],
  remove: ["prod_321"]
})
.then(({ collection }) => {
  console.log(collection)
})
```
