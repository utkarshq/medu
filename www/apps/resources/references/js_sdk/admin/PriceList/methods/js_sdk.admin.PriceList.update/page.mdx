# update

This method updates a price list. It sends a request to the 
[Update Price List](https://docs.medusajs.com/v2/api/admin#price-lists\_postpricelistsid)
API route.

## Example

```ts
sdk.admin.priceList.update("plist_123", {
  title: "My Price List",
})
.then(({ price_list }) => {
  console.log(price_list)
})
```
