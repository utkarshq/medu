# delete

This method deletes a price list. It sends a request to the
[Delete Price List](https://docs.medusajs.com/v2/api/admin#price-lists\_deletepricelistsid)
API route.

## Example

```ts
sdk.admin.priceList.delete("plist_123")
.then(({ deleted }) => {
  console.log(deleted)
})
```
