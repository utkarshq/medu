# update

This method updates a product type. It sends a request to the
[Update Product Type](https://docs.medusajs.com/api/admin#product-types\_postproducttypesid)
API route.

## Example

```ts
sdk.admin.productType.update("ptyp_123", {
  value: "Clothes"
})
.then(({ product_type }) => {
  console.log(product_type)
})
```
