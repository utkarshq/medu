# retrieveOption

This method retrieves a product's option. It sends a request to the
[Get Option](https://docs.medusajs.com/api/admin#products\_getproductsidoptionsoption\_id)
API route.

## Example

To retrieve a product option by its ID:

```ts
sdk.admin.product.retrieveOption(
  "prod_123",
  "prodopt_123"
)
.then(({ product_option }) => {
  console.log(product_option)
})
```

To specify the fields and relations to retrieve:

```ts
sdk.admin.product.retrieveOption(
  "prod_123",
  "prodopt_123",
  {
    fields: "id,title"
  }
)
.then(({ product_option }) => {
  console.log(product_option)
})
```

Learn more about the `fields` property in the [API reference](https://docs.medusajs.com/api/store#select-fields-and-relations).
