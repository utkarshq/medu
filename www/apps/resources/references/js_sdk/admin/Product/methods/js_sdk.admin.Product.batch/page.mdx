# batch

This method manages products to create, update, or delete them. It sends a request to the 
[Manage Products](https://docs.medusajs.com/api/admin#products\_postproductsbatch)
API route.

## Example

```ts
sdk.admin.product.batch({
  create: [
    {
      title: "Shirt",
      options: [{
        title: "Default",
        values: ["Default Option"]
      }],
      variants: [
        {
          title: "Default",
          options: {
            Default: "Default Option"
          },
          prices: []
        }
      ]
    }
  ],
  update: [{
    id: "prod_123",
    title: "Pants"
  }],
  delete: ["prod_321"]
})
.then(({ created, updated, deleted }) => {
  console.log(created, updated, deleted)
})
```
