# create

This method creates a product. It sends a request to the 
[Create Product](https://docs.medusajs.com/api/admin#products\_postproducts)
API route.

## Example

```ts
sdk.admin.product.create({
  title: "Shirt",
  options: [{
    title: "Default",
    values: ["Default Option"]
  }],
  variants: [
    {
      title: "Default",
      options: {
        Default: "Default Option"
      },
      prices: []
    }
  ]
})
.then(({ product }) => {
  console.log(product)
})
```
