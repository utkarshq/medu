# initiateRequest

This method creates an order edit request. It sends a HTTP request to the
[Create Order Edit](https://docs.medusajs.com/api/admin#order-edits\_postorderedits)
API route.

## Example

```ts
sdk.admin.orderEdit.initiateRequest({
  order_id: "order_123"
})
.then(({ order_change }) => {
  console.log(order_change)
})
```
