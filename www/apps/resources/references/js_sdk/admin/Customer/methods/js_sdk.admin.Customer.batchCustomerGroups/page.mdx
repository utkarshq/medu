# batchCustomerGroups

This method manages customer groups for a customer.
It sends a request to the [Manage Customers](https://docs.medusajs.com/api/admin#customers\_postcustomersidcustomergroups)
API route.

## Example

```ts
sdk.admin.customer.batchCustomerGroups("cus_123", {
  add: ["cusgroup_123"],
  remove: ["cusgroup_321"]
})
.then(({ customer }) => {
  console.log(customer)
})
```
